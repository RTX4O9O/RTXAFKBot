## Welcome

**Fake Player Plugin (FPP)** is an advanced bot-spoofing plugin for Paper 1.21+ servers.  
It spawns realistic fake players that appear in the **tab list**, **server list**, **join/leave messages**, and as physical **Mannequin entities** in the world вҖ” indistinguishable from real players at a glance.

## Quick-Start

```
1. Drop fpp.jar into your /plugins folder.
2. Restart the server вҖ” configs are generated automatically.
3. Join the server and run:  /fpp spawn 5
4. Watch five fake players appear in tab, chat, and the world.
```

## Feature Highlights

- вң… Realistic tab-list entries with proper game-mode and latency display  
- вң… Join / leave messages broadcast like real players  
- вң… Physical **Mannequin** bodies with pushback, head-tracking, and hit-sounds  
- вң… Automatic skin resolution (server-side, zero HTTP calls in `auto` mode)  
- вң… Per-player bot limits via `fpp.bot.<num>` permission nodes  
- вң… Bot **persistence** вҖ” bots leave on shutdown and rejoin after restart  
- вң… **Swap system** вҖ” bots rotate with realistic personalities and timing  
- вң… **Fake chat** вҖ” bots send configurable messages from `bot-messages.yml`  
- вң… **Database** вҖ” SQLite (default) or MySQL session history  
- вң… Full **LuckPerms** compatibility via Bukkit permission layer  
- вң… MiniMessage colour formatting throughout
